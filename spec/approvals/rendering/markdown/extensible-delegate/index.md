# mygit

Sample application that delegates unknown commands to a different executable

| Attributes       | &nbsp;
|------------------|-------------
| Version:         | 0.1.0
| Extensible:      | git

## Usage

```bash
mygit COMMAND
```

## Commands

- [push](mygit%20push) - Push to my repository
- [pull](mygit%20pull) - Pull from my repository
