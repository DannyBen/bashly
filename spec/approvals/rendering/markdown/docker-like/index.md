# docker

Docker example

| Attributes       | &nbsp;
|------------------|-------------
| Version:         | 0.1.0

## Usage

```bash
docker [OPTIONS] COMMAND
```

## Commands

- [container](docker container) - Container commands
- [image](docker image) - Image commands
- [ps](docker ps) - List containers

## Options

#### *--debug, -d*

Enable debug mode
